    Overview

    Represents a number of months passed. Used for shifting Times by a specified number of months.

    Time.local(2016, 2, 1) + 13.months # => 2017-03-01 00:00:00
Time.local(2016, 2, 29) + 2.years  # => 2018-02-28 00:00:00
